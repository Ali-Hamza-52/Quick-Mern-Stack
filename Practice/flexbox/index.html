Define a Flexbox

A flexbox is a container that allows you to arrange items on a page in a flexible way.

Flexbox Properties

The flex container properties are:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

The flex-direction Property

The flex-direction property defines in which direction the container wants to stack the flex items.
